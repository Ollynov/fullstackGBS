What is Full Stack not ?


Full Stack is NOT synonymous with
  • Web Development
  • Front End / UX Development
  • DevOps
  • Javascript...

Full Stack is about more than just programming languages

It is a methodology, an approach, a mindset.



Full Stackers...
  • Think through applications holistically
  • Employ modern engineering best practices
  • Fuse open source and proprietary tools

...to create meaningful experiences for users.
